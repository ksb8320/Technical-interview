 ### 두 개의 정수 n과 m이 주어집니다. 반복문을 사용하지 않고 별(*) 문자를 이용해 가로의 길이가 n, 세로의 길이가 m인 직사각형 형태를 출력해보세요.

``` shell
# 문제
n=5
m=9
```

```shell
n=5
m=9
print(("*"*n+'\n')*m)
```

### 다음과 같은 이차방정식이 있을 때 근을 찾는 수식을 파이썬 코드를 이용하여 출력해보시오.

``` shell
x2 +4x-21
```

```shell
a=1
b=4
c=-21

print((-b+((b**2-4*a*c)**(1/2)))/(2*a))
print((-b-((b**2-4*a*c)**(1/2)))/(2*a))
```
